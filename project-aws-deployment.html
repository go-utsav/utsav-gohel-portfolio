Dynamic Website Deployment and Hosting on AWS

Project Details

  • Virtual Private Cloud (VPC): Established a secure, isolated section of the AWS cloud for networking.
  • Internet Gateway: Enabled seamless communication between instances and the internet.
  • Security Groups: Configured robust virtual firewalls for controlling traffic to and from instances.
  • Availability Zones: Enhanced reliability and fault tolerance by distributing resources across multiple zones.
  • Public and Private Subnets: Strategically placed infrastructure components for security and performance optimization.
  • NAT Gateway: Provided secure internet access for instances in private subnets.
  • EC2 Instance Connect Endpoint: Facilitated secure connections to EC2 instances.
  • Application Load Balancer: Distributed incoming application traffic across multiple instances for high availability and fault tolerance.
  • Auto Scaling Group: Automated the management of EC2 instances to ensure optimal performance and cost-efficiency.
  • AWS Certificate Manager: Managed SSL/TLS certificates for secure communications.
  • Simple Notification Service (SNS): Enabled real-time notifications for activities within the Auto Scaling Group.
  • Route 53: Provided reliable DNS services for domain management.
  • S3 Bucket: Stored application code and assets securely.
  • Amazon RDS: Utilized a scalable, high-availability database service for storing application data.

Technologies Used
